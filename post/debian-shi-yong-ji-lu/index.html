CLI

zshell
bash默认配置文件为 .bashrc，若修改默认shell为zshell，则使用.zshrc文件为配置文件，文件位于用户文件夹内。

command

使用whereis查找软件目录
使用lsblk查看挂载硬盘
使用time测试命令执行时间

network

dns catch

sudo /etc/init.d/networking restart
//清除DNS缓存

resolvconf -u //更新DNS

nameserver

/etc/resolv.conf //DNS server 临时缓存

或者

sudo apt insatll resolvconf
/etc/resolvconf/resolv.conf.d/original 长期修改

source

dpkg -S /usr/bin/tee //查看命令所属包
sudo apt-get source coreutils //下载源码

仍未解决

3.无线网卡的固件支持问题

按网络教程查看网卡型号，使用apt安装固件

见一人显卡驱动有问题，无法进入GUI
解决方如下

sudo apt install linux-image-5.9.0-0.bpo.2-amd64 //更新内核
sudo update-grub //更新grub
sudo apt install firmware-linux-nonfree //已提前添加特别源
sudo reboot //重启
